# VidioAi Studio: Director's Cut

A professional multimodal prompt engineering suite designed for high-fidelity commercial outputs.

## Features
- **Veo Architect:** Generates motion scripts for Google Veo video generation.
- **Nano Banana 3 Pro:** Native high-fidelity image previewing.
- **Domain Logic:** Specialized engines for Automotive, Tech, and Product cinematography.
- **Single-File Architecture:** Zero-dependency React deployment.

## Usage
1. Download `index.html`.
2. Open in Chrome/Edge/Safari.
3. Requires a valid Google Gemini API Key.
